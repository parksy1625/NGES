# NGES — Nexus Growth Evaluation Standard

NGES는 AI 시스템의 **현재 성능**과 **성장 속도**를 통합 측정하는 Nexus AI 전용 평가 프레임입니다.

단순 벤치마크 점수를 넘어, 시간에 따라 시스템이 얼마나 빠르고 안정적으로 발전하는지를 수치화합니다.

## 핵심 공식

```
NGES = (현재 능력 × 0.35) + (성장 속도 × 0.45) + (구조 효율 × 0.20)
```
